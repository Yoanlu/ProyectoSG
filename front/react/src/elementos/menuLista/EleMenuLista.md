Ejemplo EleMenuLista en el que se establece un campo clave ("id") y un campo visible("text").

```js
<EleMenuLista   
    datos={[
        { id: 1, text: "Pepe" },
        { id: 2, text: "Juan" },
        { id: 3, text: "Carlos" }
    ]}  
    campoClave="id"
    campoVisible={"text"}

    />
```
